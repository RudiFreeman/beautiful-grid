## Inbox (сырые идеи)

Сюда докидываем без оценки. Не сортируется, не приоритизируется.

### Пример: поддержка RAW
- **Что:** добавить чтение CR2/CR3/NEF/ARW для прямого импорта без конвертации в JPG.
- **Зачем:** свадебные и коммерческие фотографы работают с RAW; конвертация в JPG — лишний шаг.
- **Куда (предв.):** Future (тяжёлая фича, требует `rawloader` или libraw).
- **Дата:** 2026-05-24.

### Иконка приложения — заменить placeholder на финальную
- **Что:** текущие иконки в `src-tauri/icons/` — серые PNG-заглушки (40×40×40, RGBA). Нужна финальная иконка в стиле «минимализм + сетка/композиция» (см. PRD §7).
- **Зачем:** DoD Sprint 0 допускает placeholder, но иконка влияет на первое впечатление при открытии DMG/MSI.
- **Куда (предв.):** Sprint 0 (до релиза) — до публичного v1.0 обязательно.
- **Дата:** 2026-05-24.
- **Заметки:** используй `cargo tauri icon path/to/icon.png` для генерации всех форматов из одного исходника (1024×1024 PNG).

### Крестик на фото при наведении — удалить из библиотеки / грида
- **Что:** при hover на миниатюру показывать кнопку × в углу; клик удаляет фото из Library (и соответственно из Grid).
- **Зачем:** сейчас нет способа убрать ненужное фото без повторного импорта всего набора.
- **Куда (предв.):** Sprint 1 (MVP) или sprint-1.x — базовая UX-операция.
- **Дата:** 2026-05-25.
- **Заметки:** в Library — абсолютно позиционированная кнопка `top-1 right-1`; в Grid — аналогично поверх GridCell. Удаление только из проекта, оригинал на диске не трогать (инвариант PRD §9).

### Пример: пресеты соотношений сторон холста
- **Что:** в экспорте выбрать 1:1 / 4:5 / 9:16 / A3 / A4.
- **Зачем:** под печать и под IG-форматы.
- **Куда (предв.):** Sprint 5 или раньше.
- **Дата:** 2026-05-24.
- **Заметки:** упомянуто в PRD §12 как открытый вопрос.
