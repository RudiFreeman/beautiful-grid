# Beautiful Grid — Анализ рынка и конкурентов

> **О документе:** Расширенный анализ рынка: сегментация (collage makers, DAM, IG-планировщики, moodboard-приложения, color-sort утилиты), детальный разбор 9 конкурентов (Eagle, TurboCollage, PhotoCollage, PureRef, Milanote, Preview, digiKam, Lightroom, ColorTools), позиционирование Beautiful Grid в свободной нише, ценовая стратегия и риски. ## 1. Сегментация рынка

Идея Beautiful Grid пересекается сразу с несколькими нишами. Чтобы понимать конкурентное поле, разнесём их по осям:

| Сегмент | Назначение | Где живёт пользователь |
|---|---|---|
| **A. Десктопные коллажмейкеры** | Склеить 5–30 фото в постер/коллаж по шаблону | Casual + лайт-pro: семья, блогеры, мелкий маркетинг |
| **B. Asset / mood-board менеджеры** | Хранить и сортировать тысячи референсов, делать мудборды | Дизайнеры, иллюстраторы, арт-директора |
| **C. Instagram grid-планировщики** | Превью ленты 3×N + расписание публикаций | SMM, инфлюенсеры, бренды |
| **D. Pro DAM (Digital Asset Management)** | Каталогизация, отбор, цветокор больших съёмок | Профессиональные фотографы |
| **E. Open-source фото-менеджеры** | Локальная организация, бесплатно | Linux-аудитория, прайваси-сознательные |
| **F. Color-sort утилиты** | Сортировка картинок по цвету | Энтузиасты / нишевые fotography ремесла |

Beautiful Grid в чистом виде не попадает ни в один сегмент: ближе всего к **A + B + F**, но с фотографо-ориентированным сценарием экспорта (см. §5 PRD: «экспорт с переименованием»).

## 2. Прямые и ближайшие конкуренты

### 2.1 Eagle (en.eagle.cool) — **главный косвенный конкурент**

- **Что:** локальный десктоп-органайзер ассетов (Mac/Win), 81+ форматов (JPG/PNG/PSD/RAW/MP4/SVG…), мудборды, фильтрация по цвету, теги, smart folders, browser extension.
- **Цена:** $29.95 одноразово на 2 устройства, 30-дневный триал.
- **Сильные стороны:** зрелый продукт, культовый среди дизайнеров, цветовой пикер с палитрой автоматически, локальное хранилище.
- **Слабые стороны для нашего юзкейса:** это **библиотека**, а не «компоновщик ленты/коллажа». Нет грида с экспортом одной картинки, нет сортировки именно подборки по цветовому градиенту, нет переименования при экспорте, не open-source.
- **Чем отличаемся:** Beautiful Grid — это про **компоновку конкретной подборки на экспорт**, а не каталог всей жизни.

### 2.2 TurboCollage / CollageIt / Picture Collage Maker / Movavi

- **Что:** настольные коллажмейкеры, шаблоны (mosaic, grid, mood board, polaroid, hexagon…), drag&drop, экспорт PNG/JPG.
- **Цена:** $20–$60 разово или подписка (зависит от продукта).
- **Сильные стороны:** много шаблонов «под событие», простота, давно на рынке.
- **Слабые стороны:** заточены под малое количество фото (5–30), шаблонный, не «портфолио на 300 кадров»; нет авто-сортировки по цвету; экспорт только склейкой; UX устаревший, не open-source, частично только Windows.
- **Чем отличаемся:** ориентируемся на **большие подборки фотографа** (100–500), без жёстких шаблонов, с цветовым алгоритмом и переименованием.

### 2.3 PhotoCollage (github.com/adrienverge/PhotoCollage)

- **Что:** open-source (Python + PIL), Linux-первый, авто-раскладка фото в постер, можно перетасовывать.
- **Сильные стороны:** open-source, free, MIT-подобная лицензия.
- **Слабые стороны:** UX уровня GTK 2010-х, нет Mac/Windows из коробки (нужен flatpak/python), нет цветовой сортировки, нет переименования, проект почти не развивается.
- **Чем отличаемся:** современный UI на Tauri, нативные DMG/MSI, фокус на фотографах, актуальные алгоритмы.

### 2.4 PureRef

- **Что:** free-placement (свободное расположение) reference-доска, любимая художниками/3D.
- **Цена:** pay-what-you-want, есть free.
- **Сильные стороны:** минимализм, окно «всегда сверху», молниеносный.
- **Слабые стороны:** **не сетка** — каждое изображение позиционируется руками; нет цветовой сортировки; не для финального экспорта-композиции, а для рабочего референса.
- **Чем отличаемся:** мы про регулярную сетку и финальный артефакт, а не про канвас художника.

### 2.5 Milanote / Kosmik

- **Что:** облачные визуальные мудборды/воркспейсы (Milanote — $9.99/мес.), коллаборация.
- **Сильные стороны:** командная работа, шаблоны, web-доступ.
- **Слабые стороны:** **облако** (не подходит для коммерческой съёмки), подписка, нет экспорта склееной картинки в фоторазрешении, нет цветовых алгоритмов.
- **Чем отличаемся:** строго локально, без подписок, экспорт-ориентировано.

### 2.6 Instagram-планировщики (Preview, Hopper HQ, Tailwind, GridPeek, Publer)

- **Что:** показывают, как будет выглядеть лента 3×N, поддерживают планирование публикаций.
- **Цена:** от free (Hopper HQ basic, GridPeek) до подписки $10–30/мес.
- **Сильные стороны:** дешёвые/бесплатные, интеграция с расписанием постов, мобильные приложения.
- **Слабые стороны:** жёстко привязаны к Instagram 3×N, низкое разрешение превью, нет работы с большими файлами для печати/портфолио, нет экспорта склеенного изображения как продукта, нет работы с EXIF и переименованием.
- **Чем отличаемся:** мы про **физический артефакт** (большое изображение, переименованные файлы), не про feed-управление.

### 2.7 digiKam / Shotwell / PhotoPrism (open-source DAM)

- **Что:** мощные каталогизаторы (digiKam — про-уровень, PhotoPrism — AI-based, web), open-source.
- **Сильные стороны:** бесплатные, лицензии GPL/Apache, поддержка RAW, обширные теги.
- **Слабые стороны:** **не про композицию** — про библиотеку; перегруженный UX, особенно digiKam; PhotoPrism — это web-сервер, не настольное приложение для одного юзера; не делают сетку под экспорт.
- **Чем отличаемся:** мы — узкий специализированный инструмент с одним сценарием (импорт → грид → экспорт), а не «всё для всего».

### 2.8 Lightroom Classic / Capture One / Adobe Bridge

- **Что:** профессиональные DAM с цветокором.
- **Цена:** подписки $10–30/мес.
- **Сильные стороны:** стандарт индустрии для фотографов, RAW, проявка, рейтинги.
- **Слабые стороны:** не делают красивые сетки на экспорт, нет авто-сортировки именно по цветовому градиенту, нет «склеить в одну картинку», тяжёлые, дорогие.
- **Чем отличаемся:** **дополняем**, а не заменяем. Beautiful Grid встанет в workflow ПОСЛЕ отбора в Lightroom: фотограф экспортирует JPEG-ы и приносит к нам для компоновки портфолио/ленты.

### 2.9 Color-sort утилиты (ColorTools, brandonfowler.me/sortbycolor)

- **Что:** скрипты/web-странички, которые сортируют папку картинок по цвету (HSV/HSL).
- **Сильные стороны:** делают именно то, что нам нужно, локально.
- **Слабые стороны:** **не приложения** — это CLI/web-однодневки без UI, без проекта, без drag&drop, без экспорта.
- **Чем отличаемся:** превращаем эту нишевую функцию в часть полноценного UX.

## 4. Сценарии-победители для Beautiful Grid

Места, где конкуренты явно слабы и пользователь будет искать альтернативу:

1. **Свадебный/портретный фотограф готовит превью съёмки для клиента** — нужен красивый коллаж 300 фото с цветовой гармонией. Eagle ≠ компоновка, Lightroom не сортирует по цвету, IG-планировщики — не та фракция, коллажмейкеры захлёбываются на 300 фото.
2. **Дизайнер делает мудборд для презентации** — Milanote/Eagle, но клиент хочет PNG-файл в высоком разрешении одним куском, а не ссылку на облако. Это наш экспорт.
3. **Контент-мейкер планирует визуальный ряд серии Instagram-постов** — Preview/Tailwind делают это для one-pic-per-post, но не помогают красиво раскладывать сами кадры для печати/портфолио.
4. **Open-source-сообщество фотографов на Mac/Win**, которым неприемлемо отдавать необработанные кадры в облако (NDA, коммерческая съёмка, репортажка) — Eagle закрытый, digiKam уродливый, PhotoCollage заброшенный. Свободная ниша.

## 5. Риски и угрозы

| Риск | Описание | Митигация |
|---|---|---|
| **Eagle расширится** в сторону компоновки | У Eagle большая команда, активный roadmap, могут добавить «grid layout export» | Двигаемся быстро, фокус на MIT + UX, нацеливаемся на сегмент, который Eagle не любит (фотографы, а не дизайнеры) |
| **AI-продукты** автоматизируют раскладку | Появятся «layout my photoshoot» AI-сервисы | Локальность + open-source = доверие. Возможный путь — добавить локальные ML-модели позже |
| **Подписочные SMM-планировщики** удешевят функции | Tailwind/Preview добавят grid export | Наш сегмент — не SMM, а фотографы; UX и качество экспорта будут лучше |
| **Lightroom добавит «портфолио-сетку»** | Adobe иногда добавляет «smart layouts» | Маловероятно — Adobe фокусируется на RAW/AI; даже если добавят, наш продукт бесплатный и open-source |
| **Низкий органический рост** | OSS-проектам нужен distribution | Контент-маркетинг: блог + туториалы для фотографов на YouTube, Reddit (r/photography, r/Lightroom), Hacker News-launch при v1.0 |

| Параметр | Beautiful Grid | Eagle | TurboCollage | Preview | digiKam |
|---|---|---|---|---|---|
| Цена | **Free (MIT)** + донат | $29.95 | $30–60 | $0–13/мес | Free (GPL) |
| Канал | GitHub Releases | Web + Mac/Win App Store | Web | App Stores | пакетные менеджеры/Web |
| Триал | — (всё бесплатно) | 30 дней | да | Freemium | — |

**Монетизация Beautiful Grid:** добровольные донаты через **Buy Me a Coffee + Boosty + GitHub Sponsors** (3 платформы, выбор пользователя). Без подписок, без paywall, без telemetry. Кнопка «Поддержать» доступна во всём UI (about, settings, header) и **показывается один раз вежливым модальным окном при первом запуске**.

## 7. Выводы и фокус Sprint 0

1. **Ниша подтверждена.** Beautiful Grid занимает свободную клетку в матрице конкурентов. Прямых аналогов «локальный + open-source + грид + цвет + переименование» не существует.
2. **Главный косвенный конкурент — Eagle.** Не пытаемся быть DAM. Чёткая позиция: «инструмент компоновки и экспорта **готовой подборки**, дополняющий Lightroom/Eagle».
3. **Дифференциация в маркетинге:**
   - «Open-source. Локально. MIT-лицензия — ваши кадры не покидают компьютер.»
   - «Сделано фотографом для фотографов — режим работы со 100–500 файлами не тормозит.»
   - «Авто-сортировка по цветовому градиенту — за 1 клик из хаоса в гармонию.»
4. **MVP-фокус (v1.0)** должен закрыть именно те функции, где конкуренты слабы: батч-импорт, цветовой sort, экспорт одной картинкой, грид без шаблонной жёсткости. Группы и переименование — v1.1, не растягиваем MVP.
5. **Distribution-план для запуска:**
   - GitHub README с GIF-демо.
   - Пост на r/photography + r/Lightroom + Hacker News «Show HN».
   - Видео-туториал 2-3 минуты на YouTube.
   - Product Hunt-launch при стабильной v1.0.
